### What was built
Compare a simulation's final YES/NO/NEUTRAL consensus against a live Polymarket or Manifold Markets price on the same question. Calibration data is the credibility primitive: a simulation that matched Polymarket at ±5pp is a different object than one with no external benchmark.

- `backend/app/services/calibration_service.py` — Pure-stdlib service (~250 LoC): URL validation, verdict computation (close/near/divergent), read/write calibration.json, prefetch cache reader
- `backend/tests/test_unit_calibration.py` — 30 unit tests covering all functions, edge cases, roundtrips
- `scripts/prefetch-market-calibration.sh` — Prefetch script for sandbox-safe market data fetching
